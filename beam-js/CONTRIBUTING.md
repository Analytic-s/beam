# Contributing

Thanks for helping improve `@keylightdigital/beam`.

## Scope

This repo is the public tracking script package (`beam.js`) and its docs/types.

In scope:
- Tracking script behavior and payload changes
- TypeScript declarations
- README/docs improvements
- Privacy and security hardening

Out of scope:
- Hosted dashboard/backend features
- Billing/account support

## How to contribute

1. Open an issue first for non-trivial changes.
2. Use the bug template: `.github/ISSUE_TEMPLATE/bug_report.md`.
3. Keep PRs focused and explain behavior changes clearly.
4. If script behavior changes, include before/after byte size (`wc -c beam.js`).

For hosted product issues, contact `ralph@keylightdigital.dev`.
