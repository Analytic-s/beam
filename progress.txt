## Codebase Patterns
- Production URL is `beam-privacy.com` (not `beam.keylightdigital.com` which returns 526, nor the deploy script URL `beam.keylightdigital.dev` which redirects to `beam-privacy.com`)
- Local wrangler dev skips signup/login rate limiting when IP is 'unknown' — safe to run full smoke suite locally repeatedly
- Production rate limits: 10 signups/hour/IP, 15 logins/15min/IP — run smoke suite against prod at most once per hour
- Smoke tests at `test/smoke/smoke.spec.ts` and `test/smoke/ux-audit.spec.ts`, run with `npm run test:smoke`
- `page.request.post()` (Playwright API) shares browser cookies and follows redirects — use for testing POST redirects instead of `page.evaluate(fetch)` with `redirect: 'manual'` (which returns opaque status 0 in browser context)
- `test.use()` must be at `test.describe()` scope, not inside a `test()` body — wrap in a nested describe if needed
